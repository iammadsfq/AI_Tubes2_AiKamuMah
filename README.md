### Setup and Run

#### Setup Virtual Environment
```bash
# Windows
python -m venv venv
.venv\Scripts\activate
pip install -r requirements.txt

# Mac/Linux
python3 -m venv venv
source .venv/bin/activate
pip install -r requirements.txt
```
